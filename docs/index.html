What Only This Stack Can Deliver

Local OptiQ + MTP Speed

Native multi-model support for the best OptiQ-4bit quants (Nex, Qwen3.5/3.6, Gemma-4...) with speculative decoding. Most work happens at laptop speed and low power.

Real Grok Escalation

When policy or the local auditor flags risk, the *exact* structured context is sent to real Grok (xAI). You get frontier reasoning only where it matters — and a full trace of why it was called.

Sentinel-Grade Safety

Deterministic policy (ported from Cortex Sentinel) + continuous enforcement + rollback. Hard blocks on secrets, protected paths, external network, and production deploys. MCP-Cortex contracts declare tool effects *before* they run.

Unified Auditable Traces

Every decision — local OptiQ or Grok — is recorded with digests, risk class, effects, and reasoning. Replay the entire session. Prove what happened.

The Architecture (Grok in the Loop)

Local OptiQ Layer (Nex) → Fast multi-model inference with MTP + safe sandbox tools

Sentinel Supervision (Cortex concepts) → Policy engine, continuous enforcer, rollback, approval queue

MCP-Cortex Contracts → Structured effect declarations for every tool

Grok Escalation → When REVIEW is required, rich context goes to real Grok → structured verdict + safer alternative

Unified Traces + TUI → Everything is replayable. Local vs Grok decisions are explicit.

Quick Start (Tested on Fresh Clone)

# Modern fast path git clone https://github.com/deesatzed/OptiqMTPMLX.git cd OptiqMTPMLX uv venv .venv uv pip install -e '.[server,tui,rag]' # The signature "Grok in the Loop" experience GROK_IN_LOOP=true nex agent "Build a small FastAPI hello world in the sandbox, add tests, and verify it" # Hardened daily driver (new) nex supervise --install python scripts/hardened_supervision_showpiece.py nex trace-gallery --redact --out /tmp/audit-gallery.md # Or the beautiful TUI (now with live approvals queue) ./run.sh tui
